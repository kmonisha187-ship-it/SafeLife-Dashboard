# 🩸 Blood Donation Campaign Effectiveness Analytics

A cloud-based analytics system that helps blood banks analyze campaign performance, predict donor behavior, and generate intelligent insights using AWS services, Machine Learning, and Generative AI.

## 🚀 Features

* Campaign performance analysis
* Donor response prediction
* Donor lapse prediction
* Natural language query assistant (RAG)
* Interactive dashboards and reports
* Automated email notifications

## 🛠 Technologies Used

* Python
* AWS S3
* AWS Glue
* Amazon Athena
* AWS Lambda
* Amazon SageMaker
* Amazon DynamoDB
* Amazon QuickSight
* Apache Spark (EMR)

## 📈 Machine Learning

* Donor Response Prediction Model
* Donor Lapse Prediction Model
* XGBoost algorithm with high prediction accuracy

## 🔒 Security

* SSE-S3 Encryption
* IAM Role-based Access Control
* CloudWatch Logging
